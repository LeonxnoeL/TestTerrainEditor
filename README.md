# TestTerrainEditor
Um editor de terrenos 2D utilizando o sistema de Tilemaps do Unity.
A scene primária deste projeto se chama TerrainEdit.

# Instanciando Tiles
É possível instanciar tiles com o botão esquerdo do mouse. Quando você seleciona um espaço com uma tile presente, uma outra tile será instanciada em cima dessa. Este procedimento consegue empilhar uma quantidade de aproximadamente 10 tiles.


https://user-images.githubusercontent.com/34650669/160504531-0bb44b8e-a0f3-41cc-9094-c8b995b80d28.mp4


# Removendo Tiles
É possível remover tiles instanciadas, com o botão direito do mouse. Ao selecionar um espaço com uma tile presente, o terreno será rebaixado pois terá sua tile removida.


https://user-images.githubusercontent.com/34650669/160504723-360008a5-ebdc-4e93-8275-8230b8b4f8c8.mp4


# Criando Levels Isométricos
Este projeto faz com que seja possível elaborar o design base de um level isométrico (Greyboxing) em runtime tal como aqueles presentes em títulos de sucesso como Final Fantasy Tactics, Disgaea, Tactics Ogre e etc que também adotam dessa mesma perspectiva.
